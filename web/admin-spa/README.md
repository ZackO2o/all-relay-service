# ALL Relay 管理后台 SPA

ALL Relay Service 的管理后台前端，基于 Vue 3 + TypeScript + Naive UI 构建。

## 技术栈

- Vue 3 Composition API
- TypeScript
- Naive UI
- Pinia (状态管理)
- Tailwind CSS
- ECharts (统计图表)
- Vite

## 开发

```bash
cd web/admin-spa
npm install
npm run dev
```

## 构建

```bash
npm run build
```

构建产物输出到 `dist/` 目录，部署到主服务 `web/admin/` 路径即可。
